# AGENTS.md

Project instructions for AI agents working in this repository.

## Language policy

- **Chat / conversation:** reply in the user's preferred language (currently
  Russian).
- **Everything committed to the repo is English — no exceptions.** This
  includes: git commit messages, code comments, documentation (README, docs/**,
  this file), identifiers, log/error strings, and any file content that lands in
  version control.

Rationale: the codebase and its history stay accessible to any contributor,
while day-to-day collaboration happens in the user's language.
